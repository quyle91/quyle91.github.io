Hướng dẫn Import data - lấy dữ liệu từ website khác

Wordpress plugin

https://wordpress.org/plugins/administrator-z/

Bước 1: Chuẩn bị

Bạn lên mạng tìm kiếm các trang web có nội dung liên quan đến website hiện tại. Thường mình sẽ lấy ở các trang báo và search theo danh mục của trang đó. Mẹo: search google với từ khóa: "Archives + từ khóa " để ra kết quả ngon nhé.

Bước 2: Thực hiện

Sau khi có các bài viết cần tìm, bạn vào tool bằng Dashboard-> Tool-> Administrator Z-> Import

1. Nhập link vào phần đầu tiên. trong đó có 4 sự lựa chọn
  • 1. From Post: Nhập một bài viết cụ thể
  • 2. From category: Nhập tất cả các bài viết được gọi trong một link danh mục
  • 3. From Product: Nhập một sản phẩm cụ thể vào woocommerce (yêu cầu cài plugin Woocommerce)
  • 4. From Product category: Nhập tất cả sản phẩm woocommerce từ một danh mục sản phẩm (yêu cầu cài plugin Woocommerce)

Chú ý*: Nếu là sản phẩm, Plugin hiện tại chỉ hỗ trợ nhập vào woocommerce. Còn website nguồn là mã nguồn nào ko quan trọng, miễn sao nó ko chặn là được.

2. Nhập bộ chọn theo cấu trúc Css selector

Ví dụ về tiêu đề bài viết, khi các bạn viết CSS để chọn tiêu đề bài viết như thế nào thì nhập trong này như vậy

Một số trường có dấu ↳ có nghĩa là nó sẽ dùng lại css selector của cái trên. mục đích là chọn nhiều cái con trong cái selector cha.

Trong plugin có mẫu cho theme flatsome và selector của nó. các bạn có thể tham khảo và check cho trang mẫu

3. Tiến hành import

1. Test trước một bài

2. Test cả danh mục

3. Nhấn nút Import cả danh mục

4. Các bạn check với dữ liệu mẫu nhé

Note: Cài đặt khác

Import mode
  • one by one: Khi bấm nút import theo danh mục, plugin sẽ chạy lần lượt từng cái một cho đến khi xong thì mới nhảy sang cái tiếp theo.
Content fix
  • Save images to library Ảnh tự động lưu lại gallery website
  • Use exits images in library to post content Trước khi lưu lại gallery, ảnh sẽ được check xem có chưa, nếu có rồi thì lấy luôn ảnh đó và ko lưu thêm vào nữa
  • Rewrite image name Lưu ảnh vào gallery với tên ảnh cấu trúc cố định: tên bài viết + số thứ tự của ảnh
  • Small thumbnail url fix Một số trang web dùng CDN, hoặc sửa lại size ảnh, dùng cái này để bỏ bớt phần thừa trong tên ảnh và tìm kiếm ảnh gốc. Bạn cần phải check một số ảnh trong web nguồn cho chắc chắn.
  • Content fix Phần này mục đích để bỏ một số phần mang tính bản quyền trong nội dung như link, video hoặc mô tả dưới cùng cố định của các bài nội dung
  • Search and replace content Thay đổi nội dung của web nguồn như số điện thoại, tên cửa hàng, địa chỉ, vvv... hoặc có thể remove luôn bằng cách để trống vào phần replace.
Product
  • Include entry content images to gallery Nhập luôn ảnh trong bài viết vào gallery của sản phẩm
  • Include variations images to gallery nhập ảnh đại diện của biến thể vào gallery sản phẩm
  • price lấy định dạng tiền tệ gốc. ví dụ trong cách hiển thị của USD là 99.00 $ thì cần phải nhập 2 vào đây để lấy mình số 99. nếu ko nó sẽ lấy 9900 $
